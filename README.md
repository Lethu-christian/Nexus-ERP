# Nexus-ERP | Enterprise Resource Planning

**Nexus-ERP** is a high-performance, obsidian-themed enterprise resource planning system designed for modern corporate environments in South Africa.

## 🚀 Live Demo
[View Live Project](https://nexus-erp-neon.vercel.app)

## 🛠 Tech Stack
- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS (Custom Obsidian & Electric Blue Palette)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Icons:** Lucide React
- **Interactive Components:** Custom-built for high-end corporate UX.

## ✨ Key Features
- **Dynamic Dashboard:** Real-time metrics and system health monitoring.
- **Enterprise Security:** Built with secure-by-design principles for sensitive corporate data.
- **Obsidian Design System:** A premium, dark-mode-first aesthetic (Obsidian #121212 & Electric Blue #00D2FF).
- **SEO Optimized:** Full meta-tag and JSON-LD structured data implementation for enterprise visibility.

## 📊 Business Logic
Nexus-ERP is designed to bridge the gap in affordable, high-end ERP solutions for South African firms, focusing on speed, security, and a premium user experience that rivals international standards.
